## Installation
Prerequisites:
- Tool / runtime: Python, miniconda, jupyternotebook etc.
- Package manager: pip

Quick start:
```bash
# clone repository
git clone https://github.com/AlbertJojo2313/COP_Final_Project.git
cd COP_Final_Project.git

# install dependencies (example)
pip install -r requirements.txt (need to add requirments.txt)
```
To modify the code or make changes to the project, it is much safer to create a new branch under your name
```
# To create new branch
git branch your_name

# To switch to new branch
git switch your_name
```
